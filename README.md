# brl-converter-react-native

Esse app expo faz requisições para a api https://economia.awesomeapi.com.br/json/all e mostra a cotação de USD/BRL e de EUR/BRL. Também faz login e cadastro de usuário utilizando o Firebase.

## Como rodar

1. Instale as dependências

```
npm install
```

2. Inicie o app

```
npx expo start
```
